
# KCP Share Simulator

A Streamlit-based dashboard for simulating KCP share predictions and revenue impact based on SKU price adjustments.

---

## Features
- Upload and read Excel data (`Test.xlsx`) for SKU prices and shares.
- Editable columns based on flags (e.g., `KCC` row with Y/N).
- Predict updated shares using a TensorFlow/Keras model.
- Visualize revenue changes and SKU-level performance with Plotly charts.
